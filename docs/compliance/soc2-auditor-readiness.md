# SOC 2 Auditor Readiness Plan

Owner: Mas Yasin Arafat  
Review cadence: Monthly until observation window starts  
Status: Draft, needs auditor and observation window

## Goal

Prepare zerotrust for a SOC 2 Type II audit by selecting an auditor, confirming
scope, choosing the observation period, and collecting evidence consistently.

## Scope Proposal

Suggested Trust Services Criteria:

- Security: CC6, CC7, CC8.
- Availability: A1.
- Confidentiality / Privacy: C1/P where applicable.
